# Plan 4.1: Frontend Scaffolding (Vite + React + Tailwind)

## Objective
Initialize the frontend project using Vite with React and Tailwind CSS, following a clean modular structure.

## Context
- `.gsd/SPEC.md`
- `docker-compose.yml` (port 5173 for web)

## Tasks

<task type="auto">
  <name>Init React Project</name>
  <files>
    - web/package.json
  </files>
  <action>
    - Run `npx -y create-vite@latest web --template react`.
    - Install base dependencies: `react-router-dom`, `axios`, `lucide-react`.
  </action>
  <verify>ls web/package.json</verify>
  <done>React project is initialized in `web/` directory.</done>
</task>

<task type="auto">
  <name>Install Tailwind CSS</name>
  <files>
    - web/tailwind.config.js
    - web/src/index.css
  </files>
  <action>
    - Run Tailwind init: `npm install -D tailwindcss postcss autoprefixer && npx tailwindcss init -p` inside `web/`.
    - Configure `tailwind.config.js` to include `./src/**/*.{js,jsx,ts,tsx}`.
    - Add `@tailwind` directives to `src/index.css`.
  </action>
  <verify>grep "tailwindcss" web/package.json</verify>
  <done>Tailwind CSS is configured and ready to use.</done>
</task>

<task type="auto">
  <name>Folder Structure & Base Layout</name>
  <files>
    - web/src/components/
    - web/src/pages/
    - web/src/hooks/
    - web/src/layouts/DashboardLayout.jsx
  </files>
  <action>
    - Create standard directories.
    - Implement `DashboardLayout.jsx` with a Top Navigation Header as requested in premium redesign specs.
  </action>
  <verify>ls web/src/layouts/DashboardLayout.jsx</verify>
  <done>Frontend skeleton and layouts are ready.</done>
</task>

## Success Criteria
- [ ] Vite dev server runs.
- [ ] Tailwind CSS styles apply correctly.
- [ ] Folder hierarchy follows clean architecture (pages/components/hooks/services).
